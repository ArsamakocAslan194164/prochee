Чтобы запустить код, необходимо выполнить следующие шаги:

Скопируйте код программ П1 и П2 в файлы program1.py и program2.py соответственно. Сохраните файлы в одну директорию.

Установите библиотеку numpy для Python, если её ещё не установлена. Для этого можно использовать команду pip install numpy.

Запустите программу П1, передав ей в качестве аргументов имена файлов с данными матриц А и В, а также имя файла, в который будет сохранен результат (например, python program1.py file1.txt file2.txt output.txt). Обратите внимание, что файлы должны содержать матрицы в формате, в котором их считывает функция read_data из программы П1.

Запустите программу П2, вызовите функцию test_add_matrices() для проверки корректности работы адаптера и программы П1 (например, python program2.py).

После выполнения программ П1 и П2 результат сложения матриц будет записан в файл, который был указан в программе П1 в качестве аргумента.
