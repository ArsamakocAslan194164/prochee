Как использовать код:

Необходимо сохранить код в файле с расширением .py.
Запустите программу и передайте значения входных параметров:
Аргумент 1 - название алгоритма сортировки (selection, insertion, merge);
Аргумент 2 - название файла входных данных;
Аргумент 3 - название файла выходных данных.
Например, чтобы выполнить сортировку выбором для данных из файла "input.txt" 
и записать результат в файл "output.txt", 
нужно запустить следующую команду в терминале:

python my_program.py selection input.txt output.txt
